## ⚡ Quick Troubleshooting

### Port already in use?
```bash
# Kill port 3000
npx kill-port 3000

# Or use different port
npm run dev -- -p 3001
```

### Dependencies issue?
```bash
# Clean install
rm -rf node_modules package-lock.json
npm install
```

### Build errors?
```bash
# Clear cache
rm -rf .next
npm run dev
```
